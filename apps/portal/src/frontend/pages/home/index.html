A framework for building TypeScript web apps. A framework for building TypeScript web apps.

Webstir runs on Bun and keeps TypeScript apps easy for agentic tools to read, change, test, and ship. 

$ webstir init full my-app$ webstir watch --workspace my-app$ webstir test --workspace my-app$ webstir publish --workspace my-app
