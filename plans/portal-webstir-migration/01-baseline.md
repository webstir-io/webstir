# Portal Baseline

[Back to execmap](./EXECMAP.md)

## Goal

Capture the current Docusaurus portal contract before replacing it with Webstir SSG.

## Tasks

- Build the current portal once.
- Record generated route and asset behavior that must be preserved or intentionally changed.
- Identify Docusaurus-only features currently in use.

## Constraints

- Use this document as a parity reference only; checkbox truth stays in `EXECMAP.md`.
- Preserve public URL behavior unless the migration explicitly documents and handles a change.

## Findings

- Current build command: `bun run --filter webstir-portal build`.
- Current build result: passed; Docusaurus generated static files in `apps/portal/build`.
- Current build output is ignored by `apps/portal/.gitignore`.
- Current host contract: GitHub Pages artifact deploy, `webstir.io` CNAME, `sitemap.xml`.
- Current route style: Docusaurus emits `.html` files for no-trailing-slash routes, such as `docs/tutorials/getting-started.html`.
- Current source docs: 47 Markdown files under `apps/portal/docs/**`.
- Current static assets: `static/CNAME`, `static/img/favicon.svg`, `static/img/logo.svg`, `static/img/webstir.svg`.
- Current docs syntax is mostly plain Markdown. No active MDX component usage was found in docs content; `import` and `export` hits are code examples.

## Docusaurus Features In Use

- Docs plugin with `routeBasePath: 'docs'`.
- Autogenerated sidebars for how-to, reference, and explanations.
- Explicit tutorials sidebar ordering.
- Edit URL configured for docs pages.
- Last update time enabled only when git metadata is available.
- Prism code highlighting.
- Light/dark color mode with prefers-color-scheme support.
- Navbar, footer, custom homepage, logo, favicon, and CNAME static assets.
- Broken links and broken Markdown links are warning-level in Docusaurus config.

- The current Docusaurus portal build has passed.
- The public route inventory is recorded.
- Docusaurus-specific behavior that may affect migration parity is recorded.
